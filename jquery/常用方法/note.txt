1. $()下的常用方法
  has(): 包含 内部有没有包含(span)...的标签
  not(): filter的反义词 除了...元素
  filter(): 过滤 过滤出想要的元素
  next(): 该标签紧挨着的下一个兄弟节点
  prev(): 该标签紧挨着的上一个兄弟节点
  find(): 查找 查找这个元素下的...标签
  eq(): 查找第几个符合条件的元素
  index(): 一组元素的索引，
  attr(): 操作属性

  addClass()
  removeClass()
  width():  元素自己的宽
  innerWidth():  width + padding
  outerWidth():  width + padding  + border
  outerWidth(true):  width + padding  + border + margin
  insertBefore()
  before()
  insertAfter()
  after()
  appendTo()
  append()
  prependTo()
  prepend()
  remove(): 删除节点
  on(): 可以绑定自带事件 自定义事件
  off(): 取消事件
  scrollTop(): 纵向的滚动距离
  scrollLeft(): 横向的滚动距离
